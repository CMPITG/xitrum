Introduction¶

Xitrum (“Smurf” in Vietnamese) is a lightweight Scala web framework and web server on top of Netty:

  • It fills the gap between Scalatra and Lift: more powerful than Scalatra but less complex than Lift. You can easily define RESTful routings (Scalatra style) and postbacks (Lift style).
  • Asynchronous, in the spirit of Netty.
  • Typesafe, in the spirit of Scala.
  • Stateless, by default session is stored in cookie.

Xitrum is not a full-stack web framework like Rails because it does not provide M (in MVC). However, it does provide validation feature at Action (Controller) and View. You can use any database access library you want, while still can validate user input easily.

Xitrum is inspired by Nitrogen.
