I18n¶

GNU Gettext is used.

Mo rong them tu cai nay la tot nhat: http://code.google.com/p/gettext-commons/wiki/Tutorial

Writing internationalized messages in source code¶

t("Hello World")

Extracting messages to po files¶

xgettext -ktrc:1c,2 -ktrnc:1c,2,3 -ktr -kmarktr -ktrn:1,2 -o po/keys.pot src/*.java

po files¶

You can split messages into multiple files. Xitrum will automatically merge po files in config/i18n directory.

config
  i18n
    en.po
    ja.po
    mylib
      en.po
      ja.po

Validation messages¶

jQuery Validation plugin provides i18n error messages. Xitrum automatically include the message file corresponding to the current locale.
