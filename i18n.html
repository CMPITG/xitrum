Writing internationalized messages in source code¶

t("Hello World")

Extracting messages to po files¶

xgettext -ktrc:1c,2 -ktrnc:1c,2,3 -ktr -kmarktr -ktrn:1,2 -o po/keys.pot src/*.scala

Where to save po files¶

In your SBT project:

src
  main
    resources
      i18n
        en.po
        ja.po

You can package po files in multiple JAR files. Xitrum will automatically merge them when running.

mylib.jar
  i18n
    en.po
    ja.po

another.jar
  i18n
    en.po
    ja.po

Validation messages¶

jQuery Validation plugin provides i18n error messages. Xitrum automatically include the message file corresponding to the current locale.
